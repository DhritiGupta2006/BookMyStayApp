# Book My Stay App: Hotel Booking Management System

## Project Overview
The **Book My Stay App** presents the design and implementation of a Hotel Booking Management System to illustrate the practical application of **Core Java** and fundamental data structures in real-world scenarios.

The system is developed incrementally, focusing on core logic and system behavior rather than user interface concerns. This approach enables a deep understanding of why specific data structures are essential in building scalable and maintainable software systems.

---

## Use Case 1: Application Entry & Welcome Message

### 🎯 Objective
To establish a clear and predictable starting point for the Hotel Booking application by demonstrating how a Java program begins execution and produces console output.

### 🔄 Application Flow
1. **User Action:** The actor runs the application from the command line or IDE.
2. **JVM Invocation:** The Java Virtual Machine looks for the specific method signature `public static void main(String[] args)` and invokes it.
3. **Display:** The application prints a professional welcome message, including the application name and version.
4. **Exit:** The program execution proceeds linearly and terminates after the output is displayed.
